<h3>SUBQUERY</h3>
* nested query, inner query</br>
* ()안에 조건 넣기</br>
* 문장 잘 읽고 작성!</br>
* proj_id IN (2001, 2002) => proj_id의 하나와 같다면 true 출력(OR)</br>

#

<h3>EXISTS</h3>
* 존재하면 true -> 출력</br>
* subquery의 결과가 최소 하나의 row가 있다면 true 출력</br>

#

<h3>ANY</h3>
* <> = != 다른</br>
* 하나라도 반환한다면 True</br>

#
